Mobile App ScoutU

Full-Stack Mobile App

View Code

ScoutU is a mobile application that I developed from start to its current stage of minimum viable product. ScoutU was the first full stack mobile application that I developed

The premise of this app is a platform where college coaches can meet and interact with high school athletes for recruiting purposes. ScoutU connects these types of users directly with a matching system.

I coded this application using the Flutter native app framework. The states are managed using Flutter bloc and the database is maintained via Google's Firebase technologies.
