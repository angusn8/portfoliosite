Web App Song Recommender

React Web App

View Site

This project is a song recommender built using the Spotify API and React JS. It is currently on version 1.0 and has basic features such as recommending a song based on genre and adding it to a users playlist. It also implements spotify auth to do this. I plan on continuing to work on this project and add more features in the future.
